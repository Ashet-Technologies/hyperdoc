## Building

Requires [Zig 0.15.2](https://ziglang.org/) installed.

### Build debug application

```sh-session
[user@host] hyperdoc$ zig build
```

### Build release application

```sh-session
[user@host] hyperdoc$ zig build -Drelease
```

### Run test suite

```sh-session
[user@host] hyperdoc$ zig build test
```
